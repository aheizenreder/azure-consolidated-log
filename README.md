This repository represents a test for an approach to forward logs from a azure resource in additional Log Analytics Workspace.

The setup for this test contains
* 1 subscription with deployment of a database and first Log Analytics Workspace
* 2 subscription with deployment of Log Analytics Workspace to collect database log additionaly

The points to verify with this test are:
1. The posibility of such an approach
2. Identification of requirements for the implementation of the approach
